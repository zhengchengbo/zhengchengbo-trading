## 模型总览

三层选股系统：**基本面筛选 → 事件驱动精选 → 技术面择时**

```
全A股 ─→ 第一层：基本面+赛道筛选 ─→ 第二层：事件驱动筛选 ─→ 第三层：技术面择时 ─→ 买入
```

---

## 第一层：基本面 + 赛道 + 细分方向 + 未来增长

### Step 1：范围过滤

| 条件 | 规则 |
|:----|:-----|
| 范围 | 全 A 股（主板 60/00开头、创业板 30开头、中小板） |
| 排除 | 科创板（688开头） |
| 排除 | ST、*ST |

### Step 2：赛道判定

以下 7 大赛道，全部属于国家战略方向，有明确的国务院/部委级政策文件背书，且赛道整体连续 3 年增长。**赛道不是一成不变的，每季度根据国家政策和市场行情评估调整。**

| # | 赛道 | 核心政策依据 |
|:-:|:----|:------------|
| 1 | 人工智能 / 算力 | 《新一代人工智能发展规划》、国家算力枢纽建设 |
| 2 | 半导体 / 芯片 | 国家大基金一/二/三期、《集成电路产业发展纲要》 |
| 3 | 高端制造 / 机器人 | 《中国制造2025》、《人形机器人创新发展指导意见》 |
| 4 | 新能源 / 储能 | 《"十四五"新型储能实施方案》、《2030年前碳达峰行动方案》 |
| 5 | 创新药 / 生物医药 | 《"十四五"医药工业发展规划》、《全链条支持创新药发展》 |
| 6 | 低空经济 | 《低空经济高质量发展指导意见》 |
| 7 | 信创 / 数字经济 | 《数字中国建设整体布局规划》 |

### Step 3：细分方向判定

**核心原则：既要验证过去增长，更要判断未来增长逻辑。过去增长但未来不增长的，同样排除。**

#### ✅ 纳入的细分方向

| 赛道 | 纳入的细分方向 | 未来增长依据 |
|:----|:--------------|:------------|
| AI/算力 | AI芯片、光模块/光通信、服务器/液冷、AI应用 | 全球AI资本开支年增45%+，算力需求远未见顶 |
| 半导体 | 半导体设备、半导体材料、先进封装 | 国产替代长期逻辑，渗透率低空间大 |
| 高端制造/机器人 | 人形机器人、核心零部件（丝杠/减速器/传感器）、工业母机 | 政策+资本+量产三端共振 |
| 新能源/储能 | 储能系统、锂电池（龙头）| 海外订单翻倍增长，新能源配储强制 |
| 创新药 | 创新药研发、CXO、高端医疗器械 | 商业化放量+BD出海，盈利拐点已现 |
| 低空经济 | eVTOL、工业无人机、空管系统 | 政策驱动，从0到1阶段 |
| 信创/数字经济 | 国产OS/数据库、信息安全、工业软件、云计算 | 党政信创→行业信创，持续渗透 |

#### ❌ 排除的细分方向

| 赛道 | 排除的细分方向 | 排除原因 |
|:----|:--------------|:---------|
| 新能源 | 光伏组件 | 产能严重过剩，价格战持续，未来增长逻辑不成立 |
| 新能源 | 光伏逆变器 | 增速放缓，竞争加剧 |
| 创新药 | 仿制药 | 集采持续冲击，增长停滞 |

### Step 4：主营 + 出海验证

| 条件 | 规则 | 说明 |
|:----|:-----|:-----|
| 主营验证 | 细分方向相关收入占比 ≥ 50% | 排除"挂羊头卖狗肉"的蹭概念公司 |
| 出海验证 | 海外收入占比 ≥ 20% | 有国际竞争力，不只在国内卷 |
| 数据缺失处理 | 无法获取收入构成数据的 | **宁可错过不买错**，直接排除 |

> 通过 mx-data 查询公司主营业务收入构成进行验证。

### Step 5：财务条件

| 指标 | 门槛 | 来源出处 |
|:----:|:----:|:---------|
| 毛利率 | ≥ 30% | 林园的长处：有护城河，定价权 |
| ROE（净资产收益率）| ≥ 10% | 张坤/丘栋荣的底线：能赚钱 |
| 资产负债率 | ≤ 60% | 张坤的长处：不靠借钱撑场面 |
| 净利润 | > 0 | 所有人共识：不亏损 |

## 赛道更新机制

赛道和细分方向**不是一成不变的**，需根据国家政策和市场行情定期调整。

### 更新频率
- **常规更新**：每季度一次（随季报/政策文件同步）
- **紧急更新**：重大政策出台或行业基本面突变时随时调整

### 更新流程

```
触发更新（每季度 / 重大政策 / 行业突变）
       ↓
我搜集最新信息：
  · 国务院/部委最新政策文件
  · 行业研报/机构预测
  · 龙头公司资本开支和订单情况
       ↓
我提出调整建议：
  · 新增赛道/细分方向：______
  · 移除赛道/细分方向：______
  · 维持不变
       ↓
你确认调整方案
       ↓
更新 GitHub 文档 + 同步模型配置
```

### 当前适用的场景举例
- 如果下半年国家出台「量子科技」专项政策 → 考虑新增赛道
- 如果低空经济政策推进不及预期 → 重新评估
- 如果AI资本开支增速放缓 → 调整AI细分方向权重
- 如果光伏产能出清完成 → 可能重新纳入
